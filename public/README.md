# public/

Place `sample-lease.pdf` here per PRD §15.1 (a purpose-built 10–15 page
residential lease with 2–3 red clauses, 3–4 yellow, a few green).

The landing page "Try with a sample lease →" button fetches
`/sample-lease.pdf` at runtime, so without this file the button will
surface the §11 error "Sample lease not found."
